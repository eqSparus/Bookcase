# Bookcase

Веб-приложение для знакомства с hibernate

## О проекте

Приложение "Книжная полка"

__Функции приложения__

- регистрация аккаунта;
- авторизация аккаунта;
- добавление книги;
- изменения статуса книги;
- скачивание книги;
- удаление книги;

## Технологии

- Spring Framework
- Spring Security
- Hibernate
- Tomcat
- Gradle
- PostgreSQL
- Bootstrap 5
